## Configuration Interfaces

### CTAConfig
```typescript
interface CTAConfig {
  text: string;                  // CTA text (e.g., "Full video on YouTube üëÜ")
  position: 'top' | 'bottom';   // Vertical position
  startTime?: number;            // When to show (default: auto-calculate)
  duration?: number;             // How long to show (default: 3 seconds)
  fontSize?: number;             // Font size (default: 40)
  fontColor?: string;            // Text color (default: 'white')
  backgroundColor?: string;      // Background color (optional)
  borderColor?: string;          // Border color (default: 'black')
  borderWidth?: number;          // Border width (default: 3)
}
```

### AnimatedCTAConfig
```typescript
interface AnimatedCTAConfig extends CTAConfig {
  animation: 'fade' | 'slide' | 'bounce' | 'pulse';
  arrow?: boolean;               // Show arrow pointing up
  arrowAnimation?: boolean;      // Animate arrow (bounce)
}
```

## Supported Animations

### 1. Fade
**Description:** Smooth fade-in over 1 second
**Use Case:** Professional, subtle appearance
**FFmpeg Formula:**
```
alpha='if(lt(t,57),0,if(lt(t,58),(t-57),1))'
```

### 2. Slide
**Description:** Slide in from top or bottom
**Use Case:** Dynamic entrance, catches attention
**FFmpeg Formula (Top):**
```
y='if(lt(t,57),-text_h,if(lt(t,58),100*(t-57),100))'
```

### 3. Bounce
**Description:** Bouncing effect with elastic easing
**Use Case:** Playful, energetic content
**FFmpeg Formula:**
```
y='100-20*sin(2*PI*(t-57))*exp(-3*(t-57))'
```

### 4. Pulse
**Description:** Pulsing size effect
**Use Case:** Draw continuous attention
**FFmpeg Formula:**
```
fontsize='40+5*sin(2*PI*(t-57))'
```

## Best Practices

### Readability
‚úÖ **Font Size:** 40-50 for TikTok (1080x1920)
‚úÖ **Border:** Always use borderWidth: 3 for contrast
‚úÖ **Colors:** White text with black border works best
‚úÖ **Background:** Add semi-transparent background for busy videos

### Positioning
‚úÖ **Top:** 100px from top (leaves room for profile pic)
‚úÖ **Bottom:** 100px from bottom (leaves room for captions)
‚úÖ **Horizontal:** Always centered

### Timing
‚úÖ **Default:** Last 3 seconds of video
‚úÖ **Custom:** Use startTime and duration for specific timing
‚úÖ **Duration:** 3-5 seconds is optimal

### Performance
‚úÖ **Simple CTA:** ~1-2 seconds processing time
‚úÖ **Animated CTA:** Slightly slower due to complex filters
‚úÖ **Batch Processing:** Process in parallel for speed
‚úÖ **Cleanup:** Remove temp files after processing

### Quality
‚úÖ **Audio:** Always preserved (`-c:a copy`)
‚úÖ **Video:** Re-encoding uses FFmpeg defaults
‚úÖ **High Quality:** Add `-c:v libx264 -crf 18` to command
‚úÖ **Balanced:** Add `-c:v libx264 -crf 23` to command
