Worked Example: Set Operations

Cambridge IGCSE Mathematics 0580 - Topic C1.2

Problem

Given the following two sets:

Set A = {2, 4, 6, 8, 10}
Set B = {6, 8, 10, 12, 14}

Find:

  • a) A ∩ B (the intersection)
  • b) A ∪ B (the union)
1

Understand the Question

We have two sets containing numbers. Set A contains even numbers from 2 to 10, while Set B contains even numbers from 6 to 14. Notice that some numbers appear in both sets (6, 8, and 10).

The intersection (∩) will give us elements that are in BOTH sets.
The union (∪) will give us ALL unique elements from both sets combined.
2

Find the Intersection (A ∩ B)

To find the intersection, we look for elements that appear in BOTH Set A and Set B.

Looking at Set A: {2, 4, 6, 8, 10}
Looking at Set B: {6, 8, 10, 12, 14}

The numbers 6, 8, and 10 appear in both sets.
3

Find the Union (A ∪ B)

To find the union, we combine ALL elements from both sets, but only list each unique element once (no duplicates).

From Set A: 2, 4, 6, 8, 10
From Set B: 6, 8, 10, 12, 14

Combining all unique elements in order: 2, 4, 6, 8, 10, 12, 14

Final Answers

a) A ∩ B = {6, 8, 10}
b) A ∪ B = {2, 4, 6, 8, 10, 12, 14}
