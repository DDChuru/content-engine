Introduction to Sets

1 A Set is a collection of distinct objects
2 Sets are denoted using curly braces: {a, b, c}
3 The Union (∪) combines all elements from both sets
4 The Intersection (∩) shows common elements
5 Example: A = {1, 2, 3} and B = {2, 3, 4}
6 A ∪ B = {1, 2, 3, 4}
7 A ∩ B = {2, 3}
