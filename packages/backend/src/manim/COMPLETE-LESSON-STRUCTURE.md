# Complete Sets Lesson - Full Video Structure

## 📚 Lesson Flow (Total: ~12-15 minutes)

### PART 1: Introduction (2-3 minutes) - NOT MANIM
**Tools:** Static slides (Remotion) or Gemini-generated backgrounds

1. **Branding Intro (10s)**
   - Logo animation
   - Background music starts
   - "Cambridge IGCSE Mathematics"

2. **Topic Title Slide (15s)**
   - Title: "Introduction to Sets"
   - Subtitle: "Understanding Collections and Relationships"
   - Visual: Clean, professional background

3. **Learning Objectives (45s)**
   - By the end of this lesson, you will be able to:
     * Define what a set is and identify set elements
     * Write sets using correct mathematical notation
     * Represent sets visually using Venn diagrams
     * Find the intersection (A ∩ B) of two sets
     * Find the union (A ∪ B) of two sets
     * Work with universal sets (ξ)
     * Solve problems involving set operations
     * Identify and correct errors in set notation

4. **Prerequisites (30s)**
   - What you should already know:
     * Basic number recognition (integers)
     * Understanding of "collections" or "groups"
     * Familiarity with curly braces { }
     * Basic logical thinking (AND, OR concepts)
   - Don't worry if you're new to sets - we'll build from the ground up!

5. **Lesson Roadmap (30s)**
   - What we'll cover:
     1. Set Basics & Notation
     2. Venn Diagrams (Visual Representation)
     3. Set Operations (Intersection & Union)
     4. Universal Sets
     5. Practice Problems
     6. Common Mistakes
     7. Summary & Key Takeaways

## 📝 Narration Script Structure

Each scene needs:

1. **Hook** - "In this section..."
2. **Explanation** - Core content
3. **Visual cue** - "Notice how..."
4. **Transition** - "Next, we'll explore..."

Example narration flow:
```
[INTRO]
"Welcome to Introduction to Sets! I'm excited to guide you through
one of the most fundamental topics in mathematics. Sets might seem
simple, but they're incredibly powerful - you'll use them throughout
your math journey. Let's dive in!"

[OBJECTIVES]
"By the end of this lesson, you'll confidently work with sets,
understand Venn diagrams, and solve real problems. Don't worry if
you're new to this - we're building everything from scratch!"

[Step 1 - What is a Set]
"So, what exactly IS a set? Simply put, a set is a collection of
distinct objects. Think of it like a basket containing items -
each item appears only once..."
```

## 🎨 Visual Style Guide

### Color Palette:
- **Primary:** Blue (#667eea) - Professional, trustworthy
- **Accent:** Orange (#f6ad55) - Highlight, attention
- **Success:** Green (#48bb78) - Correct answers
- **Error:** Red (#fc8181) - Mistakes, corrections
- **Set A:** Red (#ff6b6b)
- **Set B:** Green (#51cf66)
- **Intersection:** Yellow (#ffd43b)

### Typography:
- **Headings:** Sans-serif, bold, 48-60px
- **Body:** Sans-serif, regular, 24-32px
- **Math:** LaTeX rendering
- **Captions:** Gray, 20-24px

### Transitions:
- **Manim scenes:** Smooth fades (0.5s)
- **Practice questions:** Pause animations
- **Reveals:** Slide-in effects
