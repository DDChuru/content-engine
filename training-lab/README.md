# Training Lab

Central hub for AI-generated training content, SOPs, corrective actions, and educational materials.

## Content Collections

### Food Safety Training
- **Chemical Sanitation** - `projects/professional/food-safety/output/chemicals/`
- **Shopfloor Training** - `projects/professional/food-safety/output/shopfloor-training/`
- **Listeria Training** - `projects/professional/food-safety/output/farmwise-listeria-training/`
- **Food Safety Culture** - `projects/professional/food-safety/output/food-safety-culture/`

### Business Materials
- **FarmWise Proposal** - `projects/professional/food-safety/output/farmwise-proposal/`
- **iClean Presentation** - `projects/professional/food-safety/output/iclean-sanitech-presentation/`
- **Proposal Generator** - `projects/professional/food-safety/output/proposal-generator/`

### Educational Content
- **AI Changes Everything** - `output/ai-changes-everything/` (YouTube/TikTok)
- **Sets Lesson** - `output/lab1-sets/`
- **IGCSE Lessons** - `packages/backend/output/lessons/`

## Quick Start

```bash
# Start Training Lab server
cd apps/training-lab
npm run dev

# Or access via Content Engine Studio
# Training Lab workspace (shortcut: 5)
```

## Integration with Claude Code

Training Lab uses Claude Code as the central brain for:
- Generating new training modules
- Updating existing content
- Creating assessments and quizzes
- Translating to multiple languages
